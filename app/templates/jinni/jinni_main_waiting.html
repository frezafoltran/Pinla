In the meantime, curious as to how Jinni works?

There are two main aspects of songwriting Jinni implements






Sonority: We implemented an algorithm to measure how well two
sentences rhyme based on a modified version of minimum edit
distance, the phonetic representation of words and Metaphone 3 encoding.




Meaning: Our sentences are implemented by using Markov chains
in a similar way that word prediction algorithms work. We ensure
that sentences within a song match in meaning by using Word2Vec,
which is a machine learning Natural Language Processing algorithm.
